FAQ

Can I return if the size does not fit?

All items can be returned if still in their original condition and within 14 days of receipt for full refund or replacement

Can I return an item if found to be damaged on unpacking?

All items can be returned if found to be damaged on receipt. Select alternative to replace or request refund including original delivery and the cost of return

Can I return for wear and tear?

Only manufacturing defect is included if identified within 3 months of receipt. Wear and tear and improper use is excluded.

How often are new leggings added to the catalogue?

At least once a month. More frequent based on re-stocking need

Where do you buy your leggings from?

Many different sources

What payment options are available?

Prefer immediate EFT otherwise delivery will be delayed until payment has been received

What delivery options are available?

Prefer Pudo which will deliver to a drop off point.

What is the delivery time based on?

Dependent on which service is used and the payment option plus receipt. Dispatch will occur when payment has been recieved.

How is the delivery cost calculated?

Using Pudo there is a fixed charge based on size

Mainly polyester
